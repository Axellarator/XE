-ディレクトリー表示画面のPRINT 行コマンド Long File Name
　　　     オプション
             (WindowsでのP 行コマンドで実行するPRINTコマンドの
　　　　      パラメータに長いファイル名を使用)

             OPT PLFN  [off | on]

             off :FAT(8.3ファイル名)を使用
             on  :長いファイル名を使用(省略値)

         (注)-WIN版,GCC版でのみ有効.
              GCC版はWindows9xのDOS窓で実行する時のみ有効.
