= (,) 行コマンド
        .行を左右にシフトします。
        .左シフトでは行頭は削除
         マージン指定のあるファイルはマージン内のデータがシフトされ、
         右シフトではシフトアウトされます
        .x[n],xx-xx[n]のnは繰り返し数ではなくシフト桁数です。省略値はタブ桁数（省略値=4)
         xx-xxn,b.s形式はありますが xn,b.sはありません

      (1)x[n],xx-xx[n]形式

            ((     |1234567890ABCDEFG
            0002000|      7890ABCDEFG
            ((     |   4567      DEFG
            ))     |1234567890A
            0005000|12   678
            0006000|  34567890ABC
            ))     |1234567890ABCDEFG
            (8     |1234567890ABCDEFG
            )3     |1234567890ABCDEFG
            ((     |1234567890ABCDEFG
            0011000|1234567890ABCDEFG
            ((2    |1234567890ABCDEFG
            ))5    |1234567890ABCDEFG
            0014000|1234567890ABCDEFG
            ))     |1234567890ABCDEFG

      (2) xx-xx[n][,b][,s]形式

        ."２行続けて３桁シフト,１行スキップ"を指定の範囲で繰り返す

            0008000|1234567890ABCDEFG
            0009000|1234567890ABCDEFG

            0011000|1234567890ABCDEFG
            0012000|1234567890ABCDEFG

            0015000|1234567890ABCDEFG
