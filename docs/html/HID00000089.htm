= INIファイル出力

        INI     ［ファイル名]

            -"XE.INI" ファイル(ファイル名の省略値)が壊れた時に使います.
             ファイル名を指定しない /i オプションで立ち上げ(xe /i),
             INI コマンドを入力すれば 省略時の値が設定されたXE.INIファイル
             が出力されます．
       （注）VersionUpでキー割り当てが変更された場合INIファイルは再作成してください、
             古いINIファイルのままではVersionUPの変更が反映されません。
