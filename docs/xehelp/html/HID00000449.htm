(d)xelt
      linuxからtelnetしてAIX上でxeを起動する。

      0002でLANGを英語にする。AIXはSJIS,LinuxはEUCなので日本語が使えない。
      0003でキーマッピングファイル内のセクションを選択する環境変数設定。
      　　この例はRedHat V9からの telenetの場合のサンプルです。
      0004でキーマッピングファイルを指定してXEを起動。

        0001|# xe start shell for the telnet user from RedHat linux v9
        0002|export LANG=en_US
        0003|export XETERM=RH9
        0004|xe --kxeaixkey.rh9  $1 $2 $3 $4 $5 $6 $7 $8 $9
