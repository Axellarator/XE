   = シンタックスハイライト。

       SYN [ ? | ALL | RESET | ENTER | ON [lang-type] | OFF]
            ?                 : ヘルプと現在の状況を若干表示。
            ALL               : 更新の有無にかかわらず再評価を実施
            RESET             : config ファイルの再読み込み。
            ENTER             : 現在開いているファイルに "ENTERキー入力時に再評価実施" を設定。
            ON [lang-type]    : 現在開いているファイルで前回から変更があればの再評価を実施
                                lang-type はファイルの言語タイプ(拡張子)
                                lang-type のリセットは "" を指定する。
            OFF               : 現在開いているファイルのSyntaxHighlight表示を停止。

       オペランド無しは "SYN ON"で 省略値は Ctrl+s を割り当て。

       シンタックスハイライトとはキーワード、コメントなどを色を変えて表示する機能です。
       全体的な説明は(0)はじめにの該当項を参照してください。

#endif
